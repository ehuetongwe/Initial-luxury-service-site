# Luxury Service Website

A premium luxury service business website built with Next.js 14, TypeScript, TailwindCSS, and Framer Motion.

## 🏗️ Architecture

This project follows a modern, scalable architecture designed for luxury service businesses:

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript for type safety
- **Styling**: TailwindCSS for utility-first CSS
- **Animations**: Framer Motion for smooth interactions
- **State Management**: Zustand for lightweight state management
- **Icons**: Custom icon system

## 📁 Project Structure

```
luxury-service-website/
├── src/
│   ├── app/                 # Next.js App Router pages
│   ├── components/          # Reusable UI components
│   ├── data/               # Static data files (JSON)
│   ├── lib/                # Utility functions and configurations
│   ├── styles/             # Custom CSS files
│   ├── hooks/              # Custom React hooks
│   ├── context/            # React context providers
│   ├── store/              # Zustand stores
│   ├── queries/            # Data fetching hooks
│   ├── types/              # TypeScript type definitions
│   └── __tests__/          # Test files
├── public/                 # Static assets
├── .github/                # GitHub workflows
└── Configuration files
```
